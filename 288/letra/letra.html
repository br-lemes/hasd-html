1. Pai, ensina-me a viver;
Diz-me como devo andar;
Como receber poder,
Para ser qual Cristo.

Coro:
Ser igual a Cristo!
Ser igual a Cristo!
Diz-me como devo andar
Para ser qual Cristo.

2. Diz qual seja meu dever,
Ao servir-Te aqui, Senhor;
Como devo proceder,
Para ser qual Cristo.

3. Como devo eu falar,
Como demonstrar amor;
Como orar e trabalhar,
Para ser qual Cristo.
